# Recomendaciones para Entrega 2

1. **Modelos colaborativos:**
   - SVD (Singular Value Decomposition)
   - NMF (Non-negative Matrix Factorization)
   - k-Nearest Neighbors usuario–usuario, película–película

2. **Modelos basados en contenido:**
   - Regresión lineal con características demográficas + géneros
   - Gradient Boosting (XGBoost) para no-linealidades

3. **Modelos híbridos:**
   - Combinación weighted de colaborativo + contenido
   - Ensambles

4. **Validación rigurosa:**
   - Cross-validation estratificada
   - Análisis de errores (residuales, por género, por demográficos)
   - Curvas de aprendizaje
